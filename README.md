raylib UWP example
---

This project showcases how to setup and run raylib on UWP, please note it's currently only targeted at Xbox Series devices. 

## Overview

Application logic lives inside of `main.cpp::bootstrap`, a minimum application wrapper is provided by libuwp.  If you need customizations you must implement your own.

## Dependencies
- raylib
- libuwp
- glfw3-uwp
- mesa-uwp
